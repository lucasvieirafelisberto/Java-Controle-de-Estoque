# 📦 Sistema de Controle de Estoque em Java

Aplicação desenvolvida em **Java** para simular um sistema de controle de estoque, permitindo o gerenciamento de produtos e suas quantidades via terminal.

---

## 🚀 Funcionalidades

- Cadastro de produtos  
- Controle de quantidade em estoque  
- Entrada e saída de produtos  
- Listagem de itens cadastrados  
- Cálculo de valores (ex: total em estoque)  
- Interação via terminal  

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)  
- Uso de listas (`ArrayList`)  
- Estruturas condicionais (`if/else`)  
- Laços de repetição (`for`, `while`)  
- Entrada e saída de dados  
- Organização de código em classes  

---

## 🛠️ Tecnologias

- Java
